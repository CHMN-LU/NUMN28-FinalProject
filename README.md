# NUMN28 Final Project
The code for the NUMN28 final project is shown here in the FinalProjectSubmission.ipynb file, along with animations of the evolution of Burgers' equation over time. The first, EEAnim.gif, is generated by the explicit Euler method with mesh width $\Delta x=0.001$ and time step $\Delta t=0.3\Delta x$. In this animation, the wave can be seen forming a sharp jump as the peak catches up to the trough. The second, IEAnimLargeDt.gif, is generated using implicit Euler with multigrid preconditioning. Here we use $\Delta x=0.01$ for time efficiency. In this simulation, we make use of the fact that the implicit method can take much larger time steps, choosing $\Delta t=10\Delta x$. We can go even higher, but then we don't have enough frames to show the animation nicely. Notice that in this animation, the jump in the solution is smoothed. The last animation, IEAnimSmallDt.gif, uses a smaller time step of about $(4/3)\Delta x$. This is inefficient, but shows that the jump gets sharper as the time step is reduced in implicit Euler as well.

All code was written by C. Harmon using the Numpy, Scipy, and Matplotlib Python packages. Implementation choices were based on the lecture notes by Philipp Birken.
